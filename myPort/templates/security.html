Home Security System

Home Security System using Arduino, Web interface, and Email bot

The home security system was developed for a course project. The system integrates sensors to detect irregular conditions and alert users via LEDs, buzzers, and email. Additionally, an appplication called coolterm was used to connect the Arduino IDE to upload data to a csv file, then a web interface built with Streamlit was used to read the csv file, and as it updates, it allows for real-time remote monitoring, enhancing both security and sustainability.
