Online Clipboard Django App

Online Clipboard App is an app designed to help users manage and organize their tasks and items efficiently. Developed using Django, the app allows users to create personalized clipboards where they can add, view, and delete items. Each item can include text, images, and files, making it versatile for various use cases such as project management, study notes, and general task tracking.
Django (Python) was used for the backend, SQLite for the database, and HTML/CSS for the frontend, with support for file handling and media storage.
